# Module 00 – Onboarding & Environnement

## Objectifs
- Installer et configurer un environnement reproductible (uv + Makefile).
- Comprendre rapidement l’arborescence du dépôt et la documentation disponible.
- Valider la bonne exécution de `uv run python -m src` (même sans code final).

## Pré-requis
- Bases de Python et du terminal.
- Git installé localement.

## Séquence
| Étape | Contenu | Livrable |
|-------|---------|----------|
| 1 | Lecture rapide de `Doc/Sujet/` et `README.md` | Notes personnelles |
| 2 | Installation de uv + création de l’environnement | `uv.lock` généré |
| 3 | Vérification du `Makefile` (install/run/debug/lint/clean) | Commande `make install` OK |
| 4 | Ajout d’une commande `make doc` (ouvre `Doc/`) | Makefile mis à jour |
